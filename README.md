# LWJGL-2D-Engine


> Game Engine based on LWJGL using Slick 2D and Abzzezz Util.

> Look at the example to start.

## Includes:
> File Management.

> Easy to use LWJGL setup (Look at the example class).

> Screen management.

> Buttons etc.
